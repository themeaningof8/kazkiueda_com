# はじめに

このサイトは、以下の技術スタックで構築されています：

- **Astro 5.x** - 静的サイトジェネレーター
- **TypeScript** - 型安全性
- **Tailwind CSS** - スタイリング
- **Content Collections** - コンテンツ管理

## Content Collectionsの利点

1. **型安全性** - frontmatterの型チェック
2. **自動補完** - IDE支援
3. **バリデーション** - 入力値検証
4. **パフォーマンス** - 最適化されたクエリ
